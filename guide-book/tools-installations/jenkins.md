## Tool prerequisites

* OS

  * mac

## **Linux Installation:**

### centos installation

```bash
JENKINS_REPO_URI="jenkins-2.150.2-1.1"
yum install -y https://prodjenkinsreleases.blob.core.windows.net/redhat-stable/${JENKINS_REPO_URI}.noarch.rpm java
systemctl enable jenkins
systemctl start jenkins
```
