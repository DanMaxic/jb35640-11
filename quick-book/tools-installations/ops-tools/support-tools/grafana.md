# GRAFANA

## Tool prerequisites

* OS
  * mac
  * linux \( centos\ubuntu\amazon\)

## **Linux Installation:**

### centos installation

```bash
curl https://s3-us-west-2.amazonaws.com/grafana-releases/release/grafana-5.2.4-1.x86_64.rpm  --output grafana.rpm 
yum -y update 
yum -y localinstall grafana.rpm
rm grafana.rpm 
yum clean all
```
