# ANSIBLE

## Tool prerequisites

* OS
  * mac
  * linux \( centos\ubuntu\amazon\)

## **Linux Installation:**

### centos installation

```bash
sudo yum install ansible
```

**ubuntu installation**

```text
apt-get update
apt-get install software-properties-common
apt-add-repository --yes --update ppa:ansible/ansible
apt-get install ansible
```
